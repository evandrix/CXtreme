This year, we will be using Visual Basic .NET and DirectX 9 Managed for 3D game development. Although Visual Basic .NET has a bad rap for generating slow machine code, it is still one of the most powerful IDEs available. DirectX will be used both to improve performance as well as to gain access to advanced GPU features.
