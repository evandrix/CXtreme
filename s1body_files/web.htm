CXtreme, AJC's Computer Club, offers a wide range of webpage design-related developments. This includes webpage design courses in the various powerful web programming languages, such as client-side languages like HyperText Markup Language (HTML) and server-side languages PreHypertext Processor (PHP)/ My Structured Query Language (MySQL) which are can be coded at no cost to deliver high-end web content.

Moreoever, no website will be complete without multimedia features. CXtreme also offers courses in Adobe Photoshop for all your graphics needs as well as Macromedia Flash, one of the world's leading design software, for animation and a truly unique computing experience.

CXtreme is also looking into expanding this wing of the Computer Club to include website design services for anyone requiring a website to be set up for an event but needs the expertise as well as webservers for hosting websites for students. CXtreme members also make full use of this knowledge to participate in numerous website design competitions to date.
